<h1>Cadastro de Cartas Países Nivel Mestre</h1>

Este projeto foi desenvolvido como parte de um trabalho acadêmico, utilizando linguagem C na IDE Code::Blocks.
<p></p>

O objetivo principal é criar um programa que permita cadastrar duas cartas de países, inserir informações relevantes sobre cada um e, ao final, compará-los para determinar o vencedor.
<br>
![console](imagens/codigo.png)
<p></p>

Principais Funcionalidades:
<br>
* Cadastrar informações dos Países(Nome, área, população, pontos turísticos e PIB)
* Cálcular o PIB per capita do País
* Calcular a Densidade Populacional
* Calcular o Super Poder(Soma de todos os atributos dos países, e critério de comparação)
* Comparar os dois Países
<br>

Como Executar(Code::Blocks):
* Baixe ou clone o repositório e no Code::Blocks selecione abrir projeto, e procure o arquivo Cadastro-Cartas-Paises.cbp e o abra
* Depois só compilar e rodar o Código
<br>
Ou pode ultilizar sua IDE de preferência, ou utilizando o VS Code(recomendo que tenha instalado a extensão Code Runner)
